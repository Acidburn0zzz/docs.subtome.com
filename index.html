What is SubToMe?

SubToMe is a universal follow button for the open web. It can be put on any website and can be used to follow these sites using any reader.

How can I use it?

If you own a website and publish feeds on this website, you are a publisher in SubToMe's world. You should put a button on your site so that your readers can become followers! If you let them follow your site, they're more likely to come back to it.

If you're just a just a regular web user, you'll probably stumble upon SubToMe button from time to time: make sure you use them to follow the sites you care about.

If you are a developer and you have created an application that can be used to follow sites, you should support SubToMe, because it's a popular button that many sites use to be followed!
